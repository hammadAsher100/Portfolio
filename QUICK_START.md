# 🚀 Quick Start Guide

## ⚡ 5-Minute Setup

### Step 1: Add Images (Required)
```
📁 Create these files:
├── assets/hammad.jpg (your photo, 400x400px)
├── assets/favicon.png (32x32px)
├── assets/og-preview.png (1200x630px)
└── assets/projects/
    ├── cinema-schedule.png
    ├── quiz-app.png
    ├── food-delivery.png
    ├── zipit-smart.png
    └── weather-finance.png
```

### Step 2: Add Resume (Required)
```
📄 Place resume.pdf in root directory
```

### Step 3: Configure Contact Form (Optional but Recommended)
1. Go to https://formspree.io/
2. Sign up and create a form
3. Copy your form ID
4. In `index.html`, replace `YOUR_FORMSPREE_ID` with your ID

### Step 4: Deploy
```bash
git add .
git commit -m "Portfolio ready"
git push origin main
```

Then enable GitHub Pages in Settings → Pages → Select 'main' branch

## 🆘 Quick Troubleshooting

**Images not showing?**
→ Check file paths and names match exactly

**Resume not downloading?**
→ Ensure file is named exactly `resume.pdf` in root

**Form not working?**
→ Configure Formspree ID in index.html

**Need help?**
→ Check browser console (F12) for errors
