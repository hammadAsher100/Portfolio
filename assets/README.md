# Assets Directory

## Required Files

### Profile Photo
- **File**: `hammad.jpg`
- **Size**: 400x400px minimum
- **Format**: JPG or PNG
- **Description**: Your professional headshot for the About section

### Favicon
- **File**: `favicon.png`
- **Size**: 32x32px or 64x64px
- **Format**: PNG
- **Description**: Browser tab icon

### Apple Touch Icon
- **File**: `apple-touch-icon.png`
- **Size**: 180x180px
- **Format**: PNG
- **Description**: iOS home screen icon

### Open Graph Preview
- **File**: `og-preview.png`
- **Size**: 1200x630px (recommended)
- **Format**: PNG or JPG
- **Description**: Social media preview image when sharing your portfolio

## Project Screenshots

Place project screenshots in the `projects/` subdirectory:

1. **cinema-schedule.png** - Cinema Schedule Website screenshot
2. **quiz-app.png** - C# Quiz Application screenshot
3. **food-delivery.png** - Java Food Delivery Simulator screenshot
4. **zipit-smart.png** - ZipIT-Smart ticketing system screenshot
5. **weather-finance.png** - Weather & Finance ETL Dashboard screenshot

**Recommended size**: 800x500px or similar 16:10 aspect ratio
**Format**: PNG or JPG

## How to Add Images

1. Take screenshots of your projects
2. Resize them to consistent dimensions
3. Optimize them for web (use tools like TinyPNG or ImageOptim)
4. Name them exactly as listed above
5. Place them in the appropriate directories

## Temporary Placeholders

Until you add real images, the site will show broken image icons. This is expected and won't break functionality.
