## 🎨 Customization

### Update Personal Information
Edit `index.html` to update:
- Name, title, and bio
- Contact information
- Social media links
- Project details
- Education history
- Skills and certifications

### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
  --accent: #3b82f6;        /* Primary blue */
  --accent-secondary: #8b5cf6; /* Purple */
}
```

### Modify Typing Animation
Edit roles array in `script.js`:
```javascript
const roles = [
  'Software Engineering Student',
  'AI & ML Enthusiast',
  'Full-Stack Developer',
  'Data Science Enthusiast'
];
```

### Common Issues

**Links not working?**
- Check URLs are correct and repositories are public
- Search for `href="#"` in `index.html` and replace with actual URLs

**Form not submitting?**
- Sign up at [formspree.io](https://formspree.io)
- Replace `YOUR_FORMSPREE_ID` in `index.html` line ~806

**Images not loading?**
- Check file paths match exactly (case-sensitive)
- Verify files are in correct directories

**Lighthouse score low?**
- Compress images with [TinyPNG](https://tinypng.com)
- Run audit at [pagespeed.web.dev](https://pagespeed.web.dev)
