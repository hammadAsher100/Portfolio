## About

This is my personal portfolio website showcasing my projects, skills, education, and achievements. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean and performant code.

---

## ✨ Features

- 🎨 **Glassmorphism UI** — Modern dark-mode design with smooth animations
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- ♿ **Accessible** — WCAG 2.1 AA compliant with ARIA labels and keyboard navigation
- 🔍 **SEO Optimized** — JSON-LD structured data, Open Graph tags, canonical URLs
- 🌓 **Dark / Light Mode** — Theme toggle with localStorage persistence
- 🎭 **Interactive** — Typing animations, scroll effects, project filter, command palette
- 📧 **Contact Form** — Formspree-powered with AJAX submission (no page reload)
- 🏆 **Achievements** — Competition wins with gold/silver badges

## 🚀 Running Locally

```bash
git clone https://github.com/hammadAsher100/Portfolio.git
cd Portfolio
# Open index.html in your browser
```

No build step or package manager required.

---

## 🚢 Deployment

The site is deployed via **GitHub Pages** from the `main` branch.

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

Changes go live automatically within a minute.
